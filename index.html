1) The Web

The world wide web is a collection of HTML documents. HTML stands for HyperText Markup Language
and is the basis for almost every webpage. Hyperlinks are links between pages making the "world wide web" weblike.

Me <-----> Computer Browser <-----> Internet <-----> Servers

2) HTML

HTML is the "heart of the web" and is the text content that makes up a webpage. HTML is the language that arranges the
content found on a webpage, it allows for the webpage to reference other content on the website and allows for the website to
have links to other pages.

HTML is largely composed of elements i.e. <.name.>content <./name.>. An element is composed of a opening and closing
elements that tells the computer what to do with the content found in between the elements.

HTML elements are can be split into two subtypes, a) inline elements and b) block elements. Inline elements will display content
within the same line if space is avalible, while block elements segregate content into its own box and into its own line.

3)URLs

Urls are unique specifiers of the location of a webpage found within a computer network. Links contain URLs to other websites.

4) HTTPs

When the end user requests a webpage through their internet browser, the request is sent as HyperText Transfer Protocol to the server.
The server then retrieves the desired document and sends it to the browser in HTML format.
